<h1>Sistema de Gestão de Estudo</h1>
<img src=https://img.shields.io/badge/STATUS-EM_DESENVOLVIMENTO-lightgreen  alt=""/>

<h2>Objetivo</h2>
<div>Este projeto, quando concluído, será uma API que possibilitará a administração 
de atividades, exercícios e mais eficiência em momentos de estudo.
<p>Particularmente, quando eu estudo há várias abas abertas, de um pdf, do drive, do YouTube.
Nesse contexto, eu notei que esta quantidade de sites prejudicava a qualidade das 
minhas horas de estudo. Dessa forma, eu decidi desenvolver esta aplicação.</p>
<p>
Além disso, este sistema permitirá que eu aplique os conceitos aprendidos em sala de aula, 
como POO, e conheceça melhor as possibilidades da linguagem java e de frameworks 
associados a esta.
</p>
<p>Ademais, quando este for concluído pretendo disponibilizá-lo para outros estudantes.</p>
</div>
<h2>Descrição do Projeto</h2>
<div>
Linguagens: Java, HTML, CSS. <br>
Frameworks: Spring Boot.
</div>
<h3>Funcionaliades</h3>
→Armazenar atividades, livros, pdf's, entre outros. <br>
→Pequeno calendário com possibilidade de marcar provas e entregas de trabalhos. <br>
→Salvar vídeo aulas - exibir na aplicação. <br>
→Dividir tela. <br>
→Possibilidade de tocar lofy ou música clássica para concentração.
